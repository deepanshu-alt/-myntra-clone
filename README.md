
# Myntra (Clone)

Project-1: make a clone of https://www.myntra.com/ site

In this project we have tried to make a look alike clone of Myntra Website. With our efforts and the technology stack, that we have learned till now in my college from my respected professor, we were able to clone the front end with similar looks and features.

## Run Locally

Clone the project

```bash
  git clone https://github.com/imbickydutta/myntra_clone
```

## Technologies we used

In this project we have used the following technologies:

- HTML
- CSS
- JavaScript

Along with that, in order to store user data and products data we also used LocalStorage (aka browser storage).

## Libraries Used

All the image and icon links have been taken from the main website ( https://www.myntra.com/ )
